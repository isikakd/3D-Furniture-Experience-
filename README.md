 3D Furniture Experience

An interactive **3D and Augmented Reality furniture experience** that allows users to explore and place furniture in a virtual environment.
The project aims to demonstrate how **AR and 3D technologies can improve the online furniture shopping experience** by letting users visualize furniture before purchasing.

🔗 **Live Demo**

https://isikakd.github.io/3D-Furniture-Experience-/

---

## 🚀 Features

* 🧊 Interactive **3D furniture models**
* 📱 **Augmented Reality (AR) viewing**
* 🖱️ Rotate, zoom and inspect furniture objects
* 🏠 Visualize furniture in a realistic environment
* 💻 Works directly in the browser

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **WebXR / AR technologies**
* **3D models (GLTF / Web-based rendering)**

---

## 🎯 Project Purpose

Online furniture shopping often makes it difficult for users to imagine how furniture will look in their home.
This project explores how **Augmented Reality and 3D visualization** can help users better understand product size, placement, and design before making a purchase.
